# ChatGal 项目记忆

## 用户工作偏好

### ⚠️ 重要：文件删除操作
- **禁止使用 rm 命令**直接删除文件或目录
- **引导用户手动删除**：当需要删除文件时，应该：
  1. 告知用户需要删除哪些文件
  2. 说明删除原因
  3. 让用户手动执行删除操作
  4. 可以提供删除命令参考，但不直接执行

### 原因
这可能是用户出于以下考虑：
- 避免误删重要文件
- 保持对文件操作的完全控制
- 安全性和可追溯性
